## Эндпоинты

### 1. Получить список рецептов

**Метод**: `GET`
**URL**: `/recipes`

### 2. Получить рецепт по ID

**Метод**: `GET`
**URL**: `/recipes/{id}`

### 3. Создать новый рецепт

**Метод**: `POST`
**URL**: `/recipes`

### 4. Обновить рецепт

**Метод**: `PUT`
**URL**: `/recipes/{id}`

### 5. Удалить рецепт

**Метод**: `DELETE`
**URL**: `/recipes/{id}`
